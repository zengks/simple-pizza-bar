*this project has all files inside a folder called "ZENG_CHEN".

*the folder "node_modules" is not included inside this project for the purpose of faster uploading.

*required module to be installed for this project:
	-body-parser
	-express
	-hbs
	-validatorjs

*a 404.hbs page is included for any pages that aren't found

*for any inputs that don't pass input validation, the browser will show all the erros in JSON format.

*a JSON google chrome extension would make it easier to see the errors...

*all static file such as css stylesheet and pictures are all included inside the foler, "public".

*all HTML element files are included inside the folder, "views".

*all order details and delivery address are stored inside the folder, "order_data".
